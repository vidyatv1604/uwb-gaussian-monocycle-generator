# uwb-gaussian-monocycle-generator
Design, simulation, and planned PCB implementation of a 0.3–3 GHz UWB Gaussian monocycle pulse generator using AWR Microwave Office for radar and high-speed communication applications.
## Hardware Implementation (Ongoing)

A PCB layout is being designed to validate the simulated results in hardware. 
High-frequency design considerations include:

- Controlled impedance microstrip lines
- Ground plane optimization
- Minimization of parasitic inductance and capacitance
- Proper RF filtering and shielding

Hardware validation will include time-domain pulse measurement and frequency response verification.
